# Yaku-Sys: Core Financiero y ERP Transaccional

## Architecture Summary
Yaku-Sys is a comprehensive financial and transactional ERP system designed to streamline business processes and ensure accountability in financial transactions. Its architecture leverages modern technologies to create a robust and scalable solution suitable for various industries.

## Design Paradigms

### Double-entry Accounting in Database
The system implements double-entry accounting principles at the database level, ensuring every financial transaction is accurately recorded, maintaining the integrity of financial data.

### Security Architecture with RBAC
To enhance security, Yaku-Sys adopts a Role-Based Access Control (RBAC) architecture. This approach restricts access to sensitive information based on user roles, minimizing the risk of unauthorized data manipulation.

### Frontend with Strict Typing
The frontend of Yaku-Sys is developed with strict typing to ensure type safety, facilitating the detection of errors at compile time and promoting the reliability of the user interface.
